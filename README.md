# 🔋 恒功率放电模拟器 (Discharge Simulator)

基于电芯实测放电数据，模拟任意恒功率放电过程的交互工具。

## 🎯 这是什么？

这是一个**电芯放电仿真工具**，从实验数据出发，预测电芯在任意恒功率条件下的表现。

**核心能力：**
- 上传电芯的恒流放电测试数据（Excel 格式）
- 设置任意恒功率、时长、起始 SoC
- 自动模拟电压、电流、SoC 变化曲线
- 导出 CSV 数据和 PNG 图表

## 🔬 技术原理

### 二维插值算法

传统方法在电流跨越倍率曲线时会产生电压跳变。

本工具使用二维插值：
```
V = f(Q, I)
```
- Q = 已放电容量
- I = 电流

对每个容量点建立「电流→电压」插值，再在容量方向插值，确保平滑。

### 时间步进

离散化放电过程（dt=0.1 秒）：
1. 计算电流：I = P / V
2. 查表得电压：V = f(Q, I)
3. 更新容量：dQ = I × dt / 3600
4. 重复直至结束

## ❓ 常见问题

### Q: 支持其他电流倍率吗？
A: 支持！数据格式中可以有任意数量的电流倍率，程序自动识别。

### Q: 数据点不均匀怎么办？
A: 插值算法会自动处理，不要求等间距数据点。

### Q: 可以模拟恒流放电吗？
A: 当前版本仅支持恒功率。如需恒流，可直接使用原始放电数据。

### Q: 模拟结果准确吗？
A: 准确度取决于输入数据质量。二维插值确保物理平滑，但外推（超出数据范围）可能不准确。
