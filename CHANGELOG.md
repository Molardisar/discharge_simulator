## [1.0.0] - 2026-04-14

### ✨ 新增功能

- **Streamlit 交互应用**
  - 文件上传界面（支持 Excel）
  - 侧边栏参数配置（电芯容量、SoC、功率、时长）
  - 实时模拟与结果展示
  - CSV + PNG 导出功能

- **二维插值算法**
  - 实现 V = f(容量，电流) 双线性插值
  - 支持任意电流值查询（非离散倍率）
  - 自动平滑度验证

- **恒功率放电模拟**
  - 时间步进算法（dt = 0.1 秒）
  - 截止条件检查（电压 < 2.5V，电量耗尽）
  - 关键指标计算（电压变化、电流变化、SoC 变化）

- **数据可视化**
  - 三子图曲线（电压、电流、SoC）
  - 原始数据预览图
  - 高分辨率 PNG 导出（150 DPI）

- **项目文档**
  - README.md：用户使用指南
  - ARCHITECTURE.md：技术架构文档
  - DEVELOPMENT.md：开发指南
  - CHANGELOG.md：变更日志

- **跨平台支持**
  - Windows 启动脚本（run_discharge_simulator.bat）
  - Linux/macOS 兼容
  - requirements.txt 依赖管理

### 🔧 技术实现

- **核心算法**
  - `load_discharge_data()`: Excel 解析，204 个数据点
  - `create_2d_interpolator()`: 二维插值函数构建
  - `simulate_constant_power()`: 时间步进模拟引擎

- **性能优化**
  - Streamlit 数据缓存（@st.cache_data）
  - 插值函数预计算
  - 平滑度自动验证（最大单步变化 < 10mV）

- **依赖管理**
  - streamlit >= 1.32.0
  - pandas >= 2.0.0
  - scipy >= 1.10.0
  - matplotlib >= 3.7.0
  - numpy >= 1.24.0
  - openpyxl >= 3.1.0

### 📊 测试验证

**测试用例**：Reliance RS50 电芯（4.95 Ah）

**模拟条件**：
- 起始 SoC: 95%
- 恒功率：290W
- 时长：90 秒

**结果**：
- 电压：3.540V → 3.319V (Δ = 0.221V)
- 电流：81.91A → 87.39A (Δ = 5.48A)
- SoC: 95% → 52% (Δ = 43%)
- 放出容量：2.127 Ah

**平滑度**：
- 最大单步变化：0.0018V
- 平均单步变化：0.0002V
- ✓ 曲线平滑，无跳变
